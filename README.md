# FREELANCER-COMPANION-BY-DORMAMMU
Version 1.0

Работает стабильно на моде Freelancer Rebirth 7.7

1. Показывает в системах: Все порталы, контейнеры, базы.
2. Строит маршрут от системы до системы, по алгоритму Дейкстры (большиство, причем очень короткий), также по алгоритму в ширину, достаточно длинный, в случае если алгоритм Дейкстры маршрут не смог построить.
3. Ищет местоположение предметов в контейнерах и в астероидных полях.

*Часть кода написана на скорую руку, без соблюдения правил ООП, исправляется в будущем.

VS 2022 - мой авторский компаньен написанный на скорую руку (Freelancer 2003).
